## Blog app 

- This is a Blog app where users can post blogs of their choice.

----

## Features
Register
,Login
,Show others posts
,Add post
,Edit post
,Delete post
,Like post
,Comment on post
,View User Profile
,Logout

---

## Tech Stacks used
- React frontend 
- Redux state management 
- Appwrite Backend 
- Tailwind styling 
- Vite framework
